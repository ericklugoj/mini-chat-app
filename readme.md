# Mini Chat App (JAVA)

This mini chat app was created to learn how web sockets works in Java.

### Requirements
- Java 23 (or above)


## Usage

### Server:
```bash
# Go to server folder
cd src/server

# Compile server
javac Server.java

# Run server once
java Server.java
```

### Client:
```bash
# Go to client folder
cd src/client

# Compile client
javac Client.java

# Run client multiple times (different terminal tabs)
java Client.java
```
